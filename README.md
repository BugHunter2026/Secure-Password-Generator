## 🛰️ Mission Overview
In an era where the "digital void" is full of threats, a weak password is like an open airlock. The **Secure Password Generator** is a lightweight, high-performance tool designed to forge unbreakable keys. 

Unlike online generators that might log your data, this tool operates in **Total Dark Mode**—100% offline, 100% private, and 0% trace.

---

## 🛠️ Technical Specs
| Feature | Description |
| :--- | :--- |
| **Entropy Engine** | Powered by Python's `secrets` module (Cryptographically strong). |
| **Zero-Cloud** | No internet connection required. Your data never leaves your orbit. |
| **Custom UI** | Modern Dark-Space aesthetic built with `CustomTkinter`. |
| **Portability** | Single executable. No installation, no leftovers. |

---

## 🚀 Core Features
* **Industrial Randomness**: Uses hardware-level noise to generate truly unpredictable strings.
* **Granular Control**: Toggle between Uppercase, Lowercase, Numbers, and special Space-Symbols.
* **Instant Clipboard**: One-click copying to minimize the time your password is "in the air."
* **No Memory**: The tool is stateless. Once closed, the generated password is gone forever from the RAM.

---

## ⚖️ Essential Disclaimer (Read First!)
> **Warning:** This tool is a generator, not a vault. It **does not store** your passwords. It is your sole responsibility to save your generated keys in a secure password manager (like Bitwarden or KeePass). The developer is not liable for any lost access to accounts or data due to forgotten passwords.

## 🛰️ How to Launch

### For Users (Fast Entry)
1. Head over to the **[Releases](https://github.com/BugHunter2026/Secure-Password-Generator/releases)** section.
2. Download the `Secure_Password_Generator.exe`.
3. Launch and generate.

### For Developers (Source Code)
1. Clone the orbit: `git clone https://github.com/BugHunter2026/Secure-Password-Generator.git`
2. Install fuel: `pip install customtkinter`
3. Ignite: `python main.py`
